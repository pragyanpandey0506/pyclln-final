# Linear Language Models

This folder holds the active linear and linear-resistor language-model work.

Subfolders:

- `vocab16/`
  - 16-token linear baseline, direct linear-resistor one-hot trainer, and result trees
- `vocab32/`
  - 32-token linear baseline, direct/hidden-path/remodel linear-resistor trainers, launchers, and result trees

The legacy `lang_model/...` linear script paths are preserved as thin wrappers.
